# oyama_baseball_orange_team

2026年 小山少年野球団オレンジ（3年生以下）の父母会運営を支援する集計システムのリポジトリです。(暫定)

## このシステムについて

父母会幹事の運営負担を分散し、子ども全員が何かで表彰される仕組みを目指した、少年野球チーム運営の支援Webシステムです。協力当番の管理、試合・個人成績の記録、子どもの表彰、懇親会の会計までを扱います。(暫定)

## 主な機能

- **協力ログ**：試合日ごとの役割当番（スコア・配車・給水・塁審）の記録と、公平な分散のための可視化
- **試合記録・年度成績**：勝敗・得失点の記録と、グラフによる年度成績の表示
- **ティーボール個人成績**：打撃・守備・出場の記録と、各種の賞・スタンプによる表彰
- **懇親会メニュー・会計**：注文の集計、会計の閲覧、集金チェック

## 技術構成

サーバーを持たず、Googleの無料枠で完結する3層構成です。

- **フロント**：GitHub Pages（HTML / CSS / JavaScript）
- **処理**：Google Apps Script
- **データ**：Google スプレッドシート

## ページ構成

- `index_Production.html` … 協力ログ・試合記録（入力用）
- `orange_Award_Production.html` … 個人成績・表彰・年度成績（全員向け）
- `accounting_Production.html` … 懇親会の注文・会計閲覧（全員向け）
- `organizer_Production.html` … 集金チェック（幹事専用）
